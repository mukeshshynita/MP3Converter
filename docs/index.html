A) Server Requirements - top

This software requires the following:

  • Linux, Windows, or XAMPP/WAMP server
  • For commercial servers: Shared, Dedicated, and VPS hosting supported
  • Apache or IIS
  • PHP 5.3+
  • cURL and PHP cURL extension enabled
  • ffmpeg.exe (included in software distribution) for Windows & XAMPP/WAMP; FFmpeg and libmp3lame codec installed on Linux
  • That's it!

B) Software Installation - top

To install the software:

  1. Unpack/Unzip the distribution file and upload the contents to a new directory on your web server via FTP.
    • This directory should be set to the lowest permissions (chmod) possible that still enable programmatic creation, opening, and modification of the containing files.
  2. Ensure that the 'logs', 'mp3', and 'videos' folders (in your new directory), as well as software.xml, have write permissions.
    • On Linux, this generally means marking the folders/files with chmod 0777.
  3. Set the location of FFmpeg in "lib/Config.php" via the _FFMPEG constant.
    • On Windows, this is something like "/YourDirectory/ffmpeg.exe".
    • On Linux, this will generally be something like "/usr/bin/ffmpeg" or "/usr/local/bin/ffmpeg".
  4. Configure additional constants/variables in "lib/Config.php" as needed.
  5. Use your Envato/CodeCanyon purchase code to download one FREE site module (your choice).
  6. Upload your free module (and any other purchased modules) to the software's "lib/extractors/" directory.
  7. Navigate to index.php of your new directory, and start converting videos!
